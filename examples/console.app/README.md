# ic-wallet-kit console app

[![node](https://img.shields.io/badge/node-22-blue)](https://www.nodejs.org/)

---

## Local run

1. Install packages

   ```
    npm install
   ```
2. Run console app

   ```
   npm run console
   ```
3. Call login or help commands, use tab for autocomplete

   ```
      .login
   ```

   ```
      .help
   ```
